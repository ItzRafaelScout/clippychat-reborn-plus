ClippyCHAT REBORN+ - Help

Welcome to ClippyCHAT REBORN+

This application is a Windows 98 styled chat featuring classic Microsoft Agents.

Getting Started

  1. Login with your nickname and room ID.
  2. If you don't specify a room ID, you'll join the default room.
  3. Once logged in, you can chat with others in the same room.
  4. Your agent will appear on the screen and can be dragged around.

Chat Commands

You can use the following commands in the chat:

Command Description
/name [new_name] Change your nickname to [new_name]
/changeagent [agent_name] Change your agent. Available agents: bonzi, peedy, clippy

Agent Features

  • Agents are draggable - simply click and drag to move them around the screen.
  • Each agent has animations and will display your messages in a speech bubble.
  • Text-to-speech is available for messages.

Window Management

  • The chat window can be moved by dragging its title bar.
  • Use the taskbar buttons to minimize/maximize windows.
  • Window controls (minimize, maximize, close) work as expected.
