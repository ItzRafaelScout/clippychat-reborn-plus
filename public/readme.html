ClippyCHAT REBORN+ Documentation

Welcome to ClippyCHAT REBORN+

The ultimate Windows 98-styled chat experience featuring animated agents!

Basic Features

  • Animated agents (Bonzi, Peedy, Clippy)
  • Real-time chat with other users
  • Draggable agents
  • Customizable hats and accessories
  • Speech bubbles with auto-closing
  • Message replies
  • Typing indicators

Chat Commands

  • /name [nickname] - Change your display name
  • /changeagent [bonzi|peedy|clippy] - Switch your agent
  • /help - Show this documentation
  • NEW! /image [url] - Share an image (jpg, jpeg, png, gif, webp)
  • NEW! /video [youtube-url] - Embed a YouTube video
  • NEW! /poll [question] - Create a yes/no poll

Using Images and Videos

You can now share images and YouTube videos in your chat messages:

  • For images, use /image https://example.com/image.jpg
  • For YouTube videos, use /video https://youtube.com/watch?v=VIDEOID or /video https://youtu.be/VIDEOID

NEW! Poll System

Create and participate in yes/no polls:

  • Type /poll Is this the best chat app ever? to create a poll
  • Everyone in the room can vote by clicking Yes or No in the poll window
  • Polls display real-time results with colored progress bars
  • Green progress bar shows "Yes" votes percentage
  • Red progress bar shows "No" votes percentage
  • You can only vote once per poll
  • Only one active poll per room at a time

Hat System

Personalize your agent with hats and accessories:

  • Open the hat menu through the agent context menu
  • Choose from various predefined hats
  • Drag to position hats perfectly on your agent
  • Your hat settings persist across sessions

Admin Features

Admin authentication required. Contact a server administrator for access.

  • /adminpass [password] - Login as admin
  • /kick [username] - Remove a user from the chat
  • /ban [username] - Ban a user by IP address
  • NEW! /mute [username] - Prevent a user from sending messages
  • NEW! /unmute [username] - Allow a muted user to send messages again
  • NEW! /announce [message] - Send an announcement to all users
  • NEW! /clear - Clear the chat history for all users
